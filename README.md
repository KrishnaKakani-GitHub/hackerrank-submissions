## Repository structure

Solutions are organized by Interview Prep Kit topic, then problem slug. Each submission is stored as a separate numbered file:

```
<topic-folder>/
  <problem-slug>/
    submission-1.py   ← first submission
    submission-2.py   ← revised / optimized
    ...
```

**Example:**
```
Dictionaries and Hashmaps/ransom-note/submission-1.py
String Manipulation/sherlock-and-the-valid-string/submission-1.py
```
